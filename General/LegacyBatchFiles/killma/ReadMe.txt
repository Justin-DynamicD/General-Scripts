This batch file gathers invalid machine accounts to be removed from the Domain.

Run gather.bat to dump the list of machines to a .txt file called output.txt.  Next run remove.bat to actually delete all the machine accounts left in output.txt after they have been double checked.

Batch file is courtesy of Microsoft, knowledgebase:
http://support.microsoft.com/support/kb/articles/Q197/4/78.ASP
